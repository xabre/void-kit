Void Kit

Find forgotten data, orphaned app containers, and bloated caches stealing your Mac's disk space. Review everything before you delete.

Download for macOS View on GitHub

Requires macOS 13 Ventura or later. Universal binary (Apple Silicon + Intel).

What It Does

🔍

System Data Explorer

Browse caches, logs, app support, dev tools, and other system folders with size breakdowns. See exactly where your storage goes.

📦

Orphaned Container Detection

Scans ~/Library/Containers for app data left behind by uninstalled apps. Uses Launch Services and Spotlight for accurate detection.

✅

Select & Review Workflow

Check items across tabs, review your full selection, and confirm before anything is deleted. Nothing happens without your approval.

🔒

Safety Labels

Every folder is labeled Safe, Review, or Do Not Delete. Color-coded indicators help you make informed decisions at a glance.

🗑

Trash, Not Delete

Items are moved to the system Trash using macOS APIs. Changed your mind? Restore from Trash before emptying it.

🌑

Dark Native UI

Built with SwiftUI. Dark theme with alternating rows, hover highlights, and a professional file-manager aesthetic.

Safety First

Every location is classified so you know what's safe to remove.
